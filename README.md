# 🏥 MediAI – AI-Powered Healthcare Assistant

MediAI is a full-stack healthcare web application that combines modern web development with Artificial Intelligence to help patients and healthcare professionals manage medical information efficiently. The platform provides features for patient management, appointment scheduling, medical record management, and AI-powered disease prediction and medical report analysis.

---

## 🚀 Features

* User-friendly and responsive interface built with React.js
* Patient and doctor management
* Appointment booking and scheduling
* Medical record management
* AI-powered disease prediction
* Medical report analysis
* Search and filter functionality
* Complete CRUD operations
* RESTful API integration
* Responsive design for desktop and mobile devices

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/MediAI.git
```

### Navigate to the Project

```bash
cd MediAI
```

### Frontend

```bash
cd frontend
npm install
npm start
```

### Backend

```bash
cd backend
mvn spring-boot:run
```

### Machine Learning Module

```bash
cd ml-model
pip install -r requirements.txt
python app.py
```
